# Timeof | Excelencia en Relojería de Lujo

Timeof es una landing page premium para una marca de relojes de lujo, destacando la artesanía, tecnología danguardista (iNTEC) y una experiencia de usuario interactiva y cinematográfica.

## Características Principales

- **Diseño Ultra-Premium**: Estética oscura con detalles dorados y animaciones fluidas.
- **Sección iNTEC Interactiva**: Exploración detallada con un "Full-Screen Inspector" que permite ver el reloj en alta definición.
- **Cotización Online**: Formulario interactivo para solicitar modelos específicos.
- **Diseño Responsivo**: Adaptado perfectamente para dispositivos móviles y escritorio.
- **Inmersión Cinemática**: Efectos de revelado scroll y micro-interacciones de lujo.

## Tecnologías Utilizadas

- **HTML5 Semántico**
- **Vanilla CSS3** (Efectos 3D, Animaciones de hardware)
- **JavaScript Moderno** (IntersectionObserver, Client-side logic)

## Cómo empezar

Solo necesitas abrir `index.html` en cualquier navegador moderno. Para una mejor experiencia, se recomienda usar un servidor local.
